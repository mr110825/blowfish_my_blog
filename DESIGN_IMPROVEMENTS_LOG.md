## 🔄 Gitワークフロー管理

### リモート同期作業
1. **Stash作成**: 現在の作業を一時保存
   ```bash
   git stash push -m "ビジュアルデザイン改善: カスタムCSS追加とUI調整"
   ```

2. **リモート取得**: 最新のmainブランチを取得
   ```bash
   git fetch origin
   git pull origin main
   ```

3. **変更適用**: Stashした変更を再適用
   ```bash
   git stash pop
   ```

4. **コンフリクト解決**: 
   - Publicフォルダの自動生成ファイルでコンフリクト発生
   - `rm -rf public && hugo`でクリーン再生成
   - 全変更をステージング
