VSCodeでの作業効率アップのためによく使用するショートカットなどをまとめます。

## VSCodeの画面構成

| エリア | 説明 | ショートカット |
|--------|------|----------------|
| サイドバー | ファイルツリーや検索結果など | `Ctrl + B` で開閉 |
| パネル | ターミナル、出力、問題など | `Ctrl + J` で開閉 |
| ステータスバー | 最下部（文字コード、言語モード等） | - |

## ファイル操作

| 操作 | ショートカット |
|------|----------------|
| 新規ファイル | `Ctrl + N` |
| ファイルを開く | `Ctrl + O` |
| 保存 | `Ctrl + S` |
| 名前を付けて保存 | `Ctrl + Shift + S` |
| ファイルを閉じる | `Ctrl + W` |

## エディタ操作

| 操作 | ショートカット | 使いどころ |
|------|----------------|------------|
| 元に戻す | `Ctrl + Z` | 間違えたとき |
| やり直し | `Ctrl + Y` | 戻しすぎたとき |
| **行の複製** | `Alt + Shift + ↓` | 繰り返し要素を作るとき |
| **行の移動** | `Alt + ↑/↓` | コードの順番を入れ替えるとき |
| 行の削除 | `Ctrl + Shift + K` | 丸ごと消したいとき |
| コメントアウト | `Ctrl + /` | デバッグ時によく使う |

## 移動・検索系

| 操作 | ショートカット | 説明 |
|------|----------------|------|
| **クイックオープン** | `Ctrl + P` | ファイル名で素早く移動 |
| **コマンドパレット** | `Ctrl + Shift + P` | 全機能にアクセス |
| シンボルへ移動 | `Ctrl + Shift + O` | 関数/クラス一覧表示 |
| 行番号へ移動 | `Ctrl + G` | 特定の行にジャンプ |
| 定義へ移動 | `F12` | 関数の定義元を見る |
| 参照を検索 | `Shift + F12` | どこで使われているか |

## 検索・置換

| 操作 | ショートカット |
|------|----------------|
| 検索 | `Ctrl + F` |
| 置換 | `Ctrl + H` |
| **全ファイル検索** | `Ctrl + Shift + F` |
| 全ファイル置換 | `Ctrl + Shift + H` |

## マルチカーソル編集

| 操作 | ショートカット | 使いどころ |
|------|----------------|------------|
| 任意の位置にカーソル追加 | `Alt + クリック` | 飛び飛びの編集 |
| 上下に連続カーソル追加 | `Ctrl + Alt + ↑/↓` | 連続行の編集 |
| 矩形選択 | `Alt + Shift + ドラッグ` | 縦方向の選択 |
| 同じ単語を全選択 | `Ctrl + Shift + L` | 変数名一括変更 |
| 次の同じ単語を選択 | `Ctrl + D` | 1つずつ追加 |

## ターミナル連携

| 操作 | ショートカット |
|------|----------------|
| ターミナルを開く/閉じる | `` Ctrl + ` `` |
| 新しいターミナル | `` Ctrl + Shift + ` `` |
| ターミナル分割 | `Ctrl + Shift + 5` |
