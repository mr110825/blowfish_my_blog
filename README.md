# TIL Blog

日々の学びや気づきをアウトプットするTIL（Today I Learn）ブログです。

## プロジェクト構成

- **Posts** (`content/posts/`): 日々の学習記録・TIL記事
- **フレームワーク**: Hugo + Blowfishテーマ
- **デプロイ**: GitHub Pages

## コマンドまとめ

### 開発サーバー起動

```bash
# 通常起動
hugo server

# 下書き含む（draft: trueの記事も表示）
hugo server -D

# 新規記事作成
hugo new posts/<記事のタイトル>.md

# 本番用ビルド
hugo
```

## 記事構成の目安

- **何を学んだか**: 簡潔に要点をまとめる
- **どう使うか**: 実際のコード例や手順
- **参考リンク**: 参照した資料やドキュメント

シンプルに、その日学んだことを記録していきます。完璧を求めずに、継続的にアウトプットすることを重視します。
